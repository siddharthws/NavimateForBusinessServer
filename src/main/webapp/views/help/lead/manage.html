Manage Leads

  • Navimate allows you to create a lead directory. You can add new leads, remove old ones and edit information
  • Adding Leads
    • Go to Lead Management
    • Click on Add leads button
    • Fill lead details in given dialog. You can add multiple leads at a time by clicking on add button. You can also upload lead using an excel template. Template can be downloaded using the given link or here .
  • Removing Leads
    • Go to Lead Management
    • Select the leads you want to edit using the checkboxes on left
    • Select Edit from Actions Dropdown
