Managing Tasks

  • You can create new tasks, close old tasks and track the status of task using Navimate dashboard.
  • To create a task
    • Go to Task Management
    • Click on Create Button
    • Create tasks by assigning Rep / Lead / Template using dropdowns. You can add multiple tasks at once using the Add Tasks Button on top. Click on the create button at bottom of dialog to create tasks.
  • To close a Task
    • Go to Task Management
    • Select the tasks you want to close using the checkboxes on left.
    • Select Close from Actions Dropdown
